# Conta Bancária (Simulação em Python)

Projeto desenvolvido para simular operações básicas de uma conta bancária utilizando Python via terminal.

O foco deste projeto é praticar lógica de programação, estruturas de controle e organização de código, simulando regras reais de um sistema bancário.

---

## Funcionalidades

- Criar conta bancária
- Depositar valores
- Sacar valores
- Consultar saldo
- Menu interativo no terminal

---

## Tecnologias utilizadas

- Python 3
- Git
- GitHub

---

```bash
git clone https://github.com/GabrielSantanaBR/Conta_Bancaria.git
